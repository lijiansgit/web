package adler32

import "hash/adler32"

adler32包实现了Adler-32校验和算法，参见RFC 1950：

Adler-32由两个每字节累积的和组成：
s1是所有字节的累积，s2是所有s1的累积。两个累积值都取65521的余数。s1初始为1，s2初始为0。
Afler-32校验和保存为s2*65536 + s1。（最高有效字节在前/大端在前）
